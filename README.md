# Scientist Platform

## Как запустить проект

### Через Visual Studio
1. Откройте файл решения `ScientistPlatform.sln`.
2. Нажмите клавишу **F5** или кнопку **Пуск** (Start) на верхней панели.

### Через командную строку (CLI)
Для запуска из консоли убедитесь, что у вас установлен .NET SDK, и выполните следующие команды из корневой папки репозитория:

```bash
# Перейдите в папку проекта ScientistPlatform
cd ScientistPlatform

# Запустите проект
dotnet run
```
*Или запустите напрямую из корня:*
```bash
dotnet run --project ScientistPlatform/ScientistPlatform.csproj
```

## Навигация и "Вкладки"
Приложение построено на принципе **одного окна**. Переключение между основным каталогом и детальными представлениями ("второй вкладкой") происходит динамически:

*   **Открыть вторую вкладку**: 
    *   Клик по карточке статьи в списке (открывает просмотр статьи).
    *   Нажатие кнопки «Создать ссылку на статью» (открывает конструктор ссылок).
*   **Закрыть вторую вкладку**: 
    *   Нажатие кнопки «Назад к каталогу» во вторичном представлении.
    *   Клик по логотипу «Scientist Platform» (возвращает в начало).

## Главный каталог (Catalog View)

### Кнопки и элементы управления
| Элемент | Действие |
| :--- | :--- |
| **Логотип** | Возврат к списку статей | 
| **Фильтры** | Выбор режима сортировки (А-Я, дата и т.д.) | 
| **Поиск по критериям** | Открывает панель детального поиска | 
| **Кнопка "Применить"** (в критериях) | Запуск фильтрации по критерию |
| **Основной поиск** | Поиск по названию/авторам в реальном времени | 
| **Найти статьи** | Принудительное обновление списка | 
| **Кнопка настроек ** | Смена формата хранения (JSON/XML) | 

### Списки
*   **Список статей**: Основной интерактивный элемент. При клике на элемент списка открывается "вторая вкладка" с полным текстом.
*   **Издательства**: Выпадающий список в правой панели. Требуется для отправки статьи в издательство.

### Боковая панель (Действия)
*   **Отправить статью**: Требует выбора издательства и ввода ISSN. Кнопка «Отправить» активируется только при соблюдении обоих условий.
*   **Скачать статьи**: Позволяет экспортировать весь текущий список (с учетом фильтров) в выбранный формат (JSON/XML/TXT).
*   **Создать ссылку**: Переход во вкладку генерации ссылок.

---

## Просмотр статьи (Article View)
*   **Текстовые блоки**: Название, авторы и текст статьи доступны для выделения и копирования.
*   **Динамические разделы**: Блоки «Методология», «Результаты» или «Описание случая» появляются только если они заполнены в соответствии с типом статьи (Review/Research/Case Study).
*   **Кнопки экспорта**: Позволяют мгновенно сохранить текущую статью в JSON, XML или TXT.

---

## Конструктор ссылок (Citation View)
*   **Выбор статьи**: Выпадающий список всех доступных в базе публикаций.
*   **Кнопка "Сформировать"**: Активируется после выбора статьи. Выводит готовую ссылку в текстовое поле.
*   **Кнопка "Сохранить в TXT"**: Появляется/активируется только после того, как ссылка была сгенерирована.
